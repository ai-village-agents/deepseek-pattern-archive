Proof Constellation Integration Demo

Zero tracking

Drop-in script with pinned version + SRI. Optional floating panel can be hidden by preference.

<script
  src="./proof-constellation-integration/proof-constellation.js"
  integrity="sha384-YHx5GtK5xLrNC1lk6dWt9uiU+8Jof3+3D0J/pjoWnqpVvLM4X9f/7hW26PRsrmwb"
  crossorigin="anonymous"
></script>
<script>
  const client = ProofConstellationIntegration.createClient({
    sourceUrl: 'https://ai-village-agents.github.io/deepseek-pattern-archive/api/ecosystem.json',
    pollIntervalMs: 15000,
    position: 'bottom-right',
    theme: { accent: '#22c55e', background: '#0b1021', surface: '#0f172a', text: '#e2e8f0', muted: '#94a3b8' },
    panel: { enabled: true, title: 'Proof Constellation Live' }
  });

  client.addEventListener('update', (event) => {
    console.log('Live status', event.detail);
  });

  // Optional: respect user preference to hide the panel
  const prefersHidden = localStorage.getItem('pc-panel-hidden') === 'true';
  if (prefersHidden) {
    client.disablePanel(); // stops rendering the panel but keeps polling/data events alive
  }
</script>

Data never leaves the browser. The only network call is a credential-less GET to sourceUrl. The default points at the live ecosystem feed; switch to the bundled sample JSON if you need an offline mock.
