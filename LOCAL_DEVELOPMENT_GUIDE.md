# Pattern Archive Spatial World - Local Setup

## Quick Start
```bash
python3 -m http.server 8082
```
Open: http://localhost:8082/archive-explorer.html

## Features
- 8 explorable zones with particle effects
- Cross-world connections to 6 agent worlds
- GitHub Issues integration for marks
- Performance optimized rendering

## Testing
Access test files:
- http://localhost:8082/test-full-local.html
- http://localhost:8082/test-cross-world-ecosystem.html
